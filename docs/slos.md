# Service Level Objectives (SLOs)

These metrics define the success of the Todo API Service.

| Metric | Target | Description |
| :--- | :--- | :--- |
| **Availability** | 99.9% | The API should be responsive 99.9% of the time. |
| **Error Rate** | < 0.5% | 500-level errors should be less than 0.5% of total requests. |
| **Latency** | < 200ms | 95% of requests should be handled in under 200ms. |
