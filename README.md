# Todo API Service Monitoring System

# Project Overview
A containerized ASP.NET Core API (`Todo Service`) deployed on Kubernetes, monitored using Prometheus and Grafana, with automated alerting configured for Discord.

#

# One-Command Setup
To deploy the application, Prometheus, and Grafana to your cluster, run:

```bash
make deploy
```
# One-Command Teardown
To teardown the application, Prometheus, and Grafana, run:

```bash
make destroy
```
